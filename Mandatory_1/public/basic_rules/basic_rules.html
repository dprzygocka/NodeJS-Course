Basic Rules

To display content to console use method console.log().

Example:
console.log("something you want to show in the console");

Comments can be added to the code in two ways: // or between /* and */

Example:
//this is a comment
//also a comment

/* this comment allows to write
multiline comments like that */

Arithmetic operators: ( + - * / )

Example:
let sum = 2 + 4;
let subtract = 4 -2;
let divide = 4/2;
let multiply = 4*2;

Assignment operator: “=” assigns a value to a variable

Example:
let x = 2;

Comparison operators:

  • == equal to

  • === equal value and equal type

  • != not equal

  • !== not equal value or not equal type

  • > greater than

  • < less than

  • >= greater than or equal to

  • <= less than or equal to

  • ? ternary operator

Type coercion:

In order to avoid type coercion use:' ===' instead of '==.' where:

  • “=== “ compares the items based on their value and the type

  • “==” compares the items value

Data types:

  • Primitives: Number, String, Boolean, Undefined, Null, Symbol

  • Non-primitives: Object
