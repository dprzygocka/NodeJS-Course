Numbers

One type of numbers: can be written with, or without decimals

Example:
let x = 3.5
let y = 6

Adding two numbers, will return the sum.

Example:
let x = 10;
let y = 10;
let sum = x + y;
cnsole.log(sum);
output: 10

Adding a number and a string will return a string.

Example:
let x = "10";
let y = 10;
let sum = x + y;
cnsole.log(sum);
output: 1010 //string

Arithmetic with a non-numeric string will result in NaN (Not a Number).

Example:
let x = "10";
let y = 10;
let number = x / y;
cnsole.log(number);
output: NaN //string

Common methods:

  • toString():returns a number as a string.

    Example:
    let x = 20;
    console.log(x.toString());
    output: 20 //type string

  • Number() : changes its argument and returns a number.

    Example:
    const numberOne = "1.10";
    const numberTwo = "2.30";
    const sum = Number(numberOne) + Number(numberTwo)
    console.log(sum)
    output: 3.4

  • parseFloat() :changes its argument and returns a floating point number.

    Example:
    const anotherNumberOne = "1.10";
    console.log(parseFloat(anotherNumberOne));
    output: 1.10

  • parseInt() :changes its argument and returns an integer.

    Example:
    const anotherNumberOne = "1.10";
    console.log(parseInt(anotherNumberOne));
    output: 1

  • toFixed() : returns a string, with the number with a specified number of decimals.

    Example:
    const anotherNumberOne = "1.10";
    const anotherNumberTwo = "2.30";
    const sumWithDecimals = Number(anotherNumberOne) + Number(anotherNumberTwo)
    console.log(sumWithDecimals.toFixed(2))
    output: 3.40

  • toPrecision(): returns a string, with a number with a specified length.

    Example:
    const anotherNumberOne = "1.10";
    const anotherNumberTwo = "2.30";
    const sumWithDecimals = Number(anotherNumberOne) + Number(anotherNumberTwo)
    console.log(sumWithDecimals.toPrecision(2))
    output: 3.4
