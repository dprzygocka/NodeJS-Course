Hoisting

Process of moving all declarations to the top of current script or function.
Hoisting moves only declarations to the top.

Declaration Example:
var y;
Initialization Example:
var y = 5;

Using a let variable before it is declared will cause ReferenceError.

Example:
carName = "Volvo";
let carName;

Using const variable before it is declared, is a syntax error, code will not run.

Example:
carName = "Volvo";
const carName;

Allows functions to be called before they are declared.

Example:
myF(2);

function myF(x) {
return 2* x;
}

Functions defined using an expression are not hoisted.

Example:
console.log(myfunction) //comment: the variable name is hoisted, the definition isn't so it is undefined

let myfunction = function() {
console.log('cat');
};
