Arrays

  • Special type of objects.

  • Store multiple values in a single variable which can be accessed by index number.

    Example
    let array = [10,20,30];
    Console.log( array[0]);
    Output: 10

Full array can be accessed by referring to the array name.

Example
let a = [“b”, “a”, “n”]
console.log(a)
output: [ “b”, “a”, “n” ]

Array can store different types in the same array.

Example
let arr = [“car”, 2, 4.6]

Ways to create an array:

  • Array literal.

    Example
    var array_name = [item1, item2, ...];

  • JavaScript key word.

    Example
    let cars = new Array("Saab", "Volvo", "BMW");

To change the elements of an array get an element by referring to his index and assign it a new value.

Example
let array = [“yes”, “no”, “maybe”]
array[0] = 'new';
console.log(array);
output: [“yes”, “no”, “maybe”]

Arrays methods:

  • push():

    Example
    const friends = [];
    friends.push(
    {name: "Karolina", age: 24},
    {name: "Marianna", age: 22},
    {name: "Michal", age: 26});
    console.log(friends)
    output:
    [
    { name: 'Karolina', age: 24 },
    { name: 'Marianna', age: 22 },
    { name: 'Michal', age: 26 }
    ]

  • pop():

    Example
    const diet = ["tomato", "cucumber", "rocket"];
    diet.pop();
    console.log(diet);
    output: [ 'tomato', 'cucumber']

  • toString():

    Example
    const numbers = [5, 3, 2, 7, 11, 12, 0, -20, 6];
    console.log(numbers.toString())
    output: 5,3,2,7,11,12,0,-20,6

  • join():

    Example
    const numbers = [5, 3, 2, 7, 11, 12, 0, -20, 6];
    console.log(numbers.join("*"))
    output: 5*3*2*7*11*12*0*-20*6

  • shift():

    Example
    const numbers = [5, 3, 2, 7, 11, 12, 0, -20, 6];
    numbers.shift();
    console.log(numbers);
    output: [3, 2, 7, 11, 12, 0, -20, 6]

  • unhift():

    Example
    const numbers = [5, 3, 2, 7, 11, 12, 0, -20, 6];
    numbers.unshift();
    console.log(numbers);
    output: [5, 5, 3, 2, 7, 11, 12, 0, -20, 6]

  • delete key word:

    Example
    const numbers = [5, 3, 2, 7, 11, 12, 0, -20, 6];
    delete numbers[0];
    console.log(numbers);
    output: [ <1 empty item>, 3, 2, 7, 11, 12, 0, -20, 6 ]

  • splice():

    Example
    const diet = ["tomato", "cucumber", "rocket"];
    diet.splice(2,0,"hamburger","soda","pizza");
    console.log(diet);
    Output: [ 'tomato', 'cucumber', 'hamburger', 'soda', 'pizza', 'rocket' ]

  • concat():

    Example
    const lettersExpanded = ["a","b","c", "d", "e", "f", "g", "h"];
    const numbers = [5, 3, 2, 7, 11, 12, 0, -20, 6];
    console.log(lettersExpanded.concat(numbers))
    output: ['a', 'b', 'c', 'd', 'e', 'f', 'g', 'h', 5, 3, 2, 7, 11, 12, 0, -20, 6]

  • slice():

    Example
    const lettersExpanded = ["a","b","c", "d", "e", "f", "g", "h"];
    console.log(lettersExpanded.slice(2));
    output: [ 'c', 'd', 'e', 'f', 'g', 'h' ]

  • sort():

    Example
    const numbers = [5, 3, 2, 7, 1, 1, 0, -20, 6];
    console.log(numbers.sort());
    output: [-20, 0, 1, 1, 2,3, 5, 6, 7]

  • reverse():

    Example
    const numbers = [5, 3, 2, 7, 1, 1, 0, -20, 6];
    console.log(numbers.reverse());
    output: [6, -20, 0, 1, 1,7, 2, 3, 5]

Array iteration methods:

  • array.forEach(): calls a function (a callback function) once for each array element.
    Function takes 3 arguments: item value, item index, array itself.

    Example
    lettersExpanded.forEach(number => console.log(number));
    output:
    a
    b
    c
    d
    e
    f
    g
    h

  • array.map(): creates a new array by performing a function on each (non-empty) array element.
    Function takes 3 arguments: item value, item index, array itself.

    Example
    const lettersExpanded = ["a","b","c", "d", "e", "f", "g", "h"];
    const lettersExpanded2 = lettersExpanded.map(element => {return element+1} );
    console.log(lettersExpanded2)
    output: ['a1', 'b1', 'c1', 'd1', 'e1', 'f1', 'g1', 'h1']

  • array.filter(): creates a new array with array elements that passes certain test.
    Function takes 3 arguments: item value, item index, array itself.

    Example
    const lettersExpanded = ["a","b","c", "d", "e", "f", "g", "h"];
    const lettersExpanded2 = lettersExpanded.filter((number,index) => {return index % 2 != 0 ? number : ""});
    console.log(lettersExpanded2)
    output: [ 'b', 'd', 'f', 'h' ]

  • array.reduce(): runs a function on each array element to produce (reduce it to) a single value.
    Function takes 4 arguments: total (the initial value / previously returned value), item value, item index, array itself.

    Example
    const numbers = [5, 3, 2, 7, 1, 1, 0, -20, 6];
    console.log(numbers.reduce((total,value) => {return total + value }));
    output: 5

  • array.every(): check if all array values pass a test.
    Function takes 3 arguments: item value, item index, array itself.

    Example
    const numbers = [5, 3, 2, 7, 1, 1, 0, -20, 6];
    console.log(numbers.every(value => value > -22));
    output: true

  • array.some(): check if some array values pass a test.
    Function takes 3 arguments: item value, item index, array itself.

    Example
    const numbers = [5, 3, 2, 7, 1, 1, 0, -20, 6];
    console.log(numbers.some(value => value > 0));
    output: true

  • array.indexOf(item, start): searches an array for an element value and returns its position (index).
    If no item found returns -1.
    If more than one found it return first occurance.

    Example
    const significantMathNumbers = [0, 2.718, 3.14159, 1729];
    console.log(significantMathNumbers.indexOf(1729));
    output: 3

  • array.find(): returns the value of the first array element that passes a test function.
    Function takes 3 arguments: item value, item index, array itself.

    Example
    const numbers = [5, 3, 2, 7, 1, 1, 0, -20, -6];
    console.log(numbers.find(value => value < 0));
    output: -20

  • array.findIndex(): returns the index of the first array element that passes a test function.
    Function takes 3 arguments: item value, item index, array itself.

    Example
    const numbers = [5, 3, 2, 7, 1, 1, 0, -20, -6];
    console.log(numbers.findIndex(value => value < 0));
    output: 7
