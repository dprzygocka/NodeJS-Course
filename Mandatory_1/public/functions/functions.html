Functions

  • Type of function in Javascript is function but can be described as object.

  • Has propertis and methods.

  • Do not specify data types for parameters.

  • Do not perform type checking on the passed arguments.

  • Do not check the number of arguments received. missing values are set to undefined.

  • Changes to arguments are not visible outside the function.

  • Changes to object properties are visible outside the function.

  • Defined with key word: 'function'.

  • Two ways of defining functions: declaration or expression.

Function Declaration

  • Declared functions are not executed immediately.

  • They have to be called in program

Example
function myfunction(parameters) {
//code
}
Invocation:
myfunction (parameters);

Function Expression

  • Can be stored in a variable which allows the variable to be used as a function.

  • The invokation of the function is by variable name.

Example
var x = function (x, y) {
return x*y
}
Invocation:
x(x,y);

Arrow Function

  • Have short syntax where keyword function, return and curly brackets are not always needed.

  • They don't have their own 'this'.

  • Not for defining object methods.

  • Not hoisted so must be defined before they are used.

Example
const x = (x,y) => x*y;

Callback Function

  • A callback is a function passed as an argument to another function.

Example
const fetchItems = (callback, ...params) => {
callback(params);
}

fetchItems (console.log, 'foo', 1);

THIS in function

  • Global object (THIS == web browser == browser window)
    Function called without an owner object, the value of this becomes the global object

  • Invoking a Function as a Method Invoking function which is a part of an object. (THIS == objects which stores the function(object method))
