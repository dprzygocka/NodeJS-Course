Strings

Used for storing and manipulating text.

Example:
let x = "Dagmara";

Have built-in properties like length: length of a string.

Example:
let x = "Dagmara";
console.log(x.length);
output: 7

Common methods:

  • indexOf(): returns the index of the first occurrence of a specified text in a string, cannot take powerful search values (regular expressions).

    Example:
    let x = “I am learning NodeJS”;
    console.log(x.indexOf(“am”, 5));
    output: -1

  • search(): searches the position of a specified string, cannot take a second start position argument.

    Example:
    let x = “I am learning NodeJS”;
    console.log(x.indexOf(“am”));
    output: 2

  • slice(start, end): returns the extracted part in a new string.

    Example:
    let str = “apple beer banana”;
    console.log(str.slice(-11, -7));
    output: beer

  • substring(start, end): cannot accept negative indexes.

    Example:
    let str = “apple beer banana”;
    console.log(str.substring(6,10));
    output: beer

  • substr(start, length): similar to slice, second parameter specifies the length of the extracted part.

    Example:
    let str = “apple beer banana”;
    console.log(str.substr(6,4));
    output: beer

  • replace(): replaces a specified value with another value.

    Example:
    const fact = "You are learning javascript!";
    const replaced = fact.replace("javascript", "Javascript");
    console.log(replaced);
    output: You are learning Javascript!

  • toUpperCase(): converted to upper case.

    Example:
    let str = “apple beer banana”;
    console.log(str.toUpperCase());
    output: APPLE BEER BANANA

  • toLowerCase(): converted to lower case.

    Example:
    let str = “apple beer banana”;
    console.log(str.toLowerCase());
    output: apple beer banana

  • trim(): removes whitespace from both sides of a string.

    Example:
    let firstName = “ Dagmara”;
    console.log(firstName.trim());
    output: Dagmara

  • charAt(): returns the character at a specified index (position) in a string

    Example:
    let firstName = “Dagmara”;
    console.log(firstName.charAt(2));
    output: g

  • split(): convert string to an array.

    Example:
    let str = “banana”;
    console.log(str.split(“”));
    output: [ 'b', 'a', 'n', 'a', 'n', 'a' ]

Property Access:

Example:
var str = "HELLO WORLD";
console.log(str[0]);
output: H

Concatenate Strings:

  • Operation performed by using + operator.

    Example:
    console.log(“Hello” + “Dagmara”)
    output: Hello Dagmara

  • Operation performed by using string literal.

    Example:
    let firstName = “Dagmara”;
    let lastName = “Przygocka”;
    console.log(“My first name is ${firstName} and my last name is ${lastName}”); output: My first name is Dagmara and my last name is Przygocka

  • Operation performed by using method concat().

    Example:
    let firstName = “Dagmara”;
    let lastName = “Przygocka”;
    console.log(firstName.concat(“ “,lastName));
    output: Dagmara Przygocka
